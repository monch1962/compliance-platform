# cicd-gate

CI/CD compliance gate that checks your Kubernetes and Docker infrastructure against **Australian ISM and Essential Eight** standards.

## Quick Start

```bash
# Coming soon — PyPI access temporarily locked
# For now, install via Homebrew or Go:
brew tap monch1962/tap && brew install cicd-gate
# or: go install github.com/monch1962/compliance-platform/packages/cicd-gate@latest
```

Requires [conftest](https://github.com/open-policy-agent/conftest) to be installed.
